# Plantillas de Revisión de Código

## Descripción

Esta carpeta contiene los prompts utilizados en la **fase de revisión** del proceso de codificación, correspondiente al rol **"IA Revisor"** de la Guía de Codificación.

## Estructura

```
review/
├── v1/    → Plantilla genérica reutilizable (proyectos V1)
└── v2/    → Prompts reales de revisión utilizados en el proyecto CARTO (V2)
```

## Resumen de aplicación por proyecto

| Proyecto | Versión de prompt | IA Revisora |
|---|---|---|
| 1 — The Hangman Game | V1 (plantilla) | GitHub Copilot (GPT-4o mini) |
| 2 — Music Web Player | V1 (plantilla) | GitHub Copilot (GPT-4o mini) |
| 3 — MiniBalatro | V1 (plantilla) | GitHub Copilot (GPT-4o mini) |
| 4 — Cartographic Project Manager | V2 (prompts reales en `v2/`) | GitHub Copilot (GPT-5.4) |
| 5 — Tennis Tournament Manager | V2 (mismos prompts que CARTO, adaptados) | GitHub Copilot (GPT-5.4) |
