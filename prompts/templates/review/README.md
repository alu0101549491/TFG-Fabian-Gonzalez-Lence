# Code Review Templates

## Description

This directory contains the prompts used during the **review phase** of the coding process, corresponding to the **"AI Reviewer"** role in the Coding Guide.

## Structure

```
review/
├── v1/    → Generic reusable template (V1 projects)
└── v2/    → Real review prompts used in the CARTO project (V2)
```

## Application summary by project

| Project | Prompt version | AI Reviewer |
|---|---|---|
| 1 — The Hangman Game | V1 (template) | GitHub Copilot (GPT-5 mini) |
| 2 — Music Web Player | V1 (template) | GitHub Copilot (GPT-5 mini) |
| 3 — MiniBalatro | V1 (template) | GitHub Copilot (GPT-5 mini) |
