# Code Review Templates — V1

## Description

This directory contains the **generic V1 code review template**, used with **GitHub Copilot (GPT-5 mini)** as AI Reviewer in the first-generation projects of the TFG.

It corresponds to the prompt for sub-phase **2.2 "AI Reviewer"** of the Coding Guide (V1 version).

## Contents

| File | Description |
|---|---|
| `module-review-template.md` | Template for reviewing TypeScript modules and classes. Evaluates design adherence (30%), code quality (25%), requirement compliance (25%), maintainability (10%) and best practices (10%). Produces an overall score and an approval/rejection decision. |

## Projects where it was applied

|---|---|
| 1 — The Hangman Game | GitHub Copilot (GPT-5 mini) |
| 2 — Music Web Player | GitHub Copilot (GPT-5 mini) |
| 3 — MiniBalatro | GitHub Copilot (GPT-5 mini) |
