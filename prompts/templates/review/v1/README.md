# Plantillas de Revisión de Código — V1

## Descripción

Esta carpeta contiene la **plantilla genérica de revisión de código V1**, utilizada con **GitHub Copilot (GPT-4o mini)** como IA Revisora en los proyectos de primera generación del TFG.

Corresponde al prompt de la subfase **2.2 "IA Revisor"** de la Guía de Codificación (versión V1).

## Contenido

| Fichero | Descripción |
|---|---|
| `module-review-template.md` | Plantilla para la revisión de módulos y clases TypeScript. Evalúa adherencia al diseño (30%), calidad de código (25%), cumplimiento de requisitos (25%), mantenibilidad (10%) y buenas prácticas (10%). Emite una puntuación global y una decisión de aprobación/rechazo. |

| 2 — Music Web Player | GitHub Copilot (GPT-4o mini) |
| 3 — MiniBalatro | GitHub Copilot (GPT-4o mini) |
