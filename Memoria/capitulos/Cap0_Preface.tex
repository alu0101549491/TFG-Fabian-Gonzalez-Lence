\chapter*{Prefacio} 
\addcontentsline{toc}{chapter}{Prefacio}
Todo el código escrito para este trabajo sigue los principios del libro \textit{Clean Code} de Martin \cite{Martin:2009:Clean} y se ajusta a las guías de estilo de Google\footnote{\href{https://google.github.io/styleguide/}{Google Style Guide}}, mientras que los fragmentos de código provenientes de otros trabajos se mantienen esencialmente sin modificaciones. La decisión de escribir código siguiendo el estilo de Google es completamente arbitraria; lo verdaderamente importante al elegir un estilo es ser coherente en su uso: \textit{``Lo último que queremos es añadir más complejidad al código fuente escribiéndolo con un batiburrillo de estilos individuales diferentes.'' - Robert C. Martin}.\\

Todo el código escrito para este proyecto está disponible públicamente en el repositorio de GitHub dedicado a este trabajo \cite{URL::TFG}.\\

Asimismo, cada listado de código incluye un enlace a su fuente original y, en el caso del código del presente trabajo, habrá un enlace directo al fichero correspondiente en el repositorio del trabajo, etiquetado como \textit{``Ver en GitHub''}.
